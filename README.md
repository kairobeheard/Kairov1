# Kairo — A Voice Sanctuary

A Minimalist, anonymous voice-sharing platform where people speak, listen, and leave lighter.

## Features
- **Total Anonymity**: No profiles, no usernames, no history.
- **Voice-First**: Focused on the raw sound of human connection.
- **Sanctuary Wall**: Listen to voices left by others in the sanctuary.
- **Waitlist**: Integrated with Supabase for early access management.

## Tech Stack
- **Frontend**: HTML5, Vanilla CSS, Vanilla JavaScript.
- **Backend**: Supabase (Database & Storage).
- **Animations**: CSS Keyframes, Canvas Particle System.

## Setup
1. Clone the repository.
2. Run a local server (e.g., `npx serve` or Live Server).
3. Ensure Supabase environment variables are configured in `js/main.js`.
